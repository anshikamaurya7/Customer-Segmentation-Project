## 🌟 Project Story

Have you ever wondered why companies send different offers to different customers?

Not every customer behaves the same way. Some spend frequently, some save money, while others only purchase occasionally. Businesses use customer segmentation to understand these differences and create personalized marketing strategies.

In this project, I applied **K-Means Clustering**, an unsupervised machine learning algorithm, to group customers based on their **Annual Income** and **Spending Score**. The goal was to uncover hidden patterns in customer behavior and transform raw data into meaningful business insights.

---

## 🎯 Problem Statement

How can businesses identify different types of customers and target them more effectively?

Using customer data, this project segments customers into distinct groups, helping organizations understand:

* High-value customers
* Budget-conscious customers
* Potential loyal customers
* Customers requiring targeted engagement
